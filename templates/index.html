Discovery Configuration

Queries routing protocol tables on each discovered device. Adds L3-only neighbors (e.g. WAN peers) not visible via CDP/LLDP.

Collects ARP tables from each discovered device. Hosts are shown as a count badge on each node and in an expandable side panel — not added as graph nodes.

ℹ️ How it works

  • Connects to seed device via SSH
  • Runs "show cdp neighbors detail" and "show lldp neighbors detail"
  • Extracts neighbor information (hostname, IP, interfaces, platform)
  • Automatically detects device types using YAML patterns
  • Recursively discovers neighbors (routers and switches only)
  • Displays topology tree with interface mappings and IPs
